This inspection suggests to replace null check with Objects/Stream static method. Example: 

if (lBrace == null) {
  SelectionManager.registerError(initializer, infos);
} else {
  SelectionManager.registerError(lBrace, infos);
}

can be replaced with


  SelectionManager.registerError(Objects.requireNonNullElse(lbrace, initializer), infos);
